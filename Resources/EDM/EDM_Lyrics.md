# EDM Lyrics Playbook

## Mission
Write concise, rhythmic lyric fragments that magnify build tension and drop impact.

## Best Practices
- Use short, high-contrast phrases.
- Place emotional trigger line before drop.
- Design hooks that survive repetition and chopping.

## LLM Workflow
- Generate one-line hook bank (20 options).
- Provide pre-drop script + post-drop mantra variants.
- Add phonetic-friendly versions for vocal processing.

## Evaluation Rubric
- Crowd chant potential
- Clarity in loud mix contexts
- Build/drop alignment
