# External Songwriting Guide Integrations (Curated for LLM Use)

This file distills practical songwriting guidance from external educational sources and open repositories into reusable writing patterns.

## Sources used

1. Berklee Online course page: **Lyric Writing: Tools and Strategies** (Pat Pattison).
2. Ableton Classroom: **Lesson 1: Melody Writing with MIDI** (Theory Hacks).
3. NSAI workshop post: **Building Better Songwriting Habits**.
4. GitHub repository: **zoejane/music_ai_prompt** (prompt patterns for creation workflows).

> Note: This is a paraphrased synthesis for practice and model prompting, not a verbatim copy of source materials.

## A) Lyric Craft System (Berklee-aligned)

### Core principles
- Generate ideas first, edit second.
- Use deliberate control of rhyme, rhythm, and line length to shape momentum.
- Create clearer contrast between verse, chorus, and bridge by changing prosody and density.
- Revise drafts with a focused pass for word precision and emotional intent.

### Practical exercises
1. **Object writing sprint (10 minutes)**
   - Pick one object (e.g., motel key, torn ticket, cracked mug).
   - Write sensory details only (sight/sound/smell/touch/taste + memory).
   - Highlight 3 lines that could become lyric anchors.

2. **Line-length contrast drill**
   - Verse: mostly longer lines (8–12 syllables).
   - Chorus: shorter, repeatable lines (4–8 syllables).
   - Bridge: switch pattern to create perceived "new section" even over similar harmony.

3. **Rhyme motion drill**
   - Start with loose/slant rhyme in verse for narrative flow.
   - Tighten rhyme in pre/chorus for memorability.
   - Add internal rhyme only where groove needs acceleration.

## B) Melody Ideation System (Ableton classroom adapted)

### Constraints that increase creativity
- Write one melody with only white keys.
- Write a second melody with only black keys.
- Limit each phrase to 3 notes before expanding.

### Development prompts
- Reorder the same three notes across different rhythms.
- Build a two-note groove and vary rests before pitch.
- Move a 4-note shape up/down to create sequence and anticipation.

### Session format (20–30 min)
1. 5 min: improvise motif options.
2. 10 min: pick one motif and create A/B phrase.
3. 10 min: arrange 4-bar idea into verse and chorus variants.

## C) Songwriting Habit Engine (NSAI-aligned)

### Weekly cadence
- 3 writing sessions minimum (even 20 minutes each).
- 1 co-writing or feedback touchpoint weekly.
- 1 review session to tag strongest hooks and unfinished ideas.

### Anti-block protocol
- When stuck, switch task type instead of stopping:
  - lyric -> melody,
  - melody -> title bank,
  - production -> topline rewrite.
- Track streaks and small wins, not only completed songs.

### Goal stack
- **Input goals**: writing minutes, idea count, co-write count.
- **Output goals**: finished demos, pitch-ready songs, releases.

## D) Prompt Pattern Library (inspired by open repo workflows)

### 1. Theme expansion
"Given title `<TITLE>` and emotion `<EMOTION>`, produce 15 chorus angles: 5 conversational, 5 poetic, 5 high-energy chant styles."

### 2. Multi-genre transfer
"Rewrite this hook for Pop, Country, Hip-Hop, Rock, and EDM while preserving core meaning and adjusting diction and rhythm per genre norms."

### 3. Prosody check
"Score each line from 1-10 for singability, stress alignment, and vowel openness on high notes. Suggest revisions that improve performance feel."

### 4. Rewrite escalator
"Create three rewrites of this verse: (A) more concrete imagery, (B) more internal rhyme, (C) fewer words and stronger cadence."

## E) Integration Blueprint for this repository

Use these external patterns as a recurring content template:
1. **Concept**: one technique (e.g., line-length contrast).
2. **Genre variant**: how it changes in pop/country/hip-hop/etc.
3. **Prompt block**: LLM-ready instruction.
4. **Evaluation rubric**: pass/fail checks for generated output.

Suggested rubric fields:
- Hook clarity (1-5)
- Emotional specificity (1-5)
- Prosody alignment (1-5)
- Section contrast (1-5)
- Memorability (1-5)
