# Hip-Hop Structure Playbook

## Mission
Balance replayable hooks with storytelling progression and performance dynamics.

## Baseline Form
Intro (4-8) → Hook (8) → Verse (16) → Hook (8) → Verse (16) → Hook (8) → Bridge/Feature (8-16) → Final Hook/Outro.

## Best Practices
- Establish identity in first 10 seconds (tag, phrase, motif).
- Ensure Verse 2 adds new information.
- Use beat switches only if narrative or energy arc justifies them.

## LLM Workflow
- Generate timeline with bar counts + content purpose.
- Add social clip segment recommendation (8-15 sec).
- Provide clean and explicit versions with preserved cadence.

## Evaluation Rubric
- Hook stickiness
- Verse progression quality
- Arrangement momentum
## Objective
Balance replayability (strong hooks) with narrative momentum (verses that progress).

## Common Form
- Intro: 4–8 bars
- Hook: 8 bars
- Verse 1: 16 bars
- Hook: 8 bars
- Verse 2: 16 bars
- Hook: 8 bars
- Bridge / breakdown / feature: 8–16 bars
- Final hook/outro: 8–16 bars

## Structural Principles
- Start quickly; dead intros reduce retention.
- Make each verse add new perspective, stakes, or detail.
- Use beat switches or dropouts only when they serve the story.

## LLM Planning Guidance
Ask for:
- Bar map with cadence intent by section.
- Hook wording variants for repetition without staleness.
- Optional feature verse placement logic.
- Performance notes (ad-libs, doubles, call-response moments).

## Quality Checklist
- Does Verse 2 reveal something new?
- Is the hook phrasing short enough for memorability?
- Are section lengths consistent with intended platform/runtime?
