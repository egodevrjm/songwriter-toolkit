# Hip-Hop Flow Playbook

## Mission
Engineer cadence as rhythm design: words should feel percussive, intentional, and character-driven.

## Genre DNA
- Pocket control matters more than rhyme density alone.
- Flow should evolve in 4-bar phrases.

## Best Practices
- Mark stressed syllables before writing full bars.
- Alternate textures: tight multis → sparse punchline bar.
- Use breathable punctuation points every 2-4 bars.
- Match cadence shape to beat subdivisions (triplet/straight/swing).

## LLM Workflow
- Generate 16 bars with stress markers.
- Produce 2 cadence variants on identical lyric content.
- Return internal rhyme maps and breath marks.
- Add ad-lib callouts for performance energy.

## Anti-Patterns
- Constant double-time with no contour.
- End-rhyme dependency without internal architecture.

## Evaluation Rubric
- Pocket lock
- Clarity at target BPM
- Punchline emphasis
# Hip-Hop Flows Playbook

## Objective
Create cadence that feels rhythmic, intentional, and emotionally aligned with the beat.

## Core Principles
- Treat flow as drum programming with words.
- Alternate density: dense internal rhyme passages vs. breathable pockets.
- Place key words on downbeats/backbeats for emphasis.
- Use rhyme families, not single end-rhyme chains.

## Flow Design Patterns
- Straight pocket (grid-aligned clarity).
- Syncopated pocket (off-grid tension).
- Double-time burst (energy spike before hook).
- Half-time drag (heavier emotional weight).

## LLM Flow Guidance
Ask the model for:
- 16-bar drafts with stressed syllables marked.
- Two cadence versions over the same lyrics.
- Internal rhyme map and multisyllable options.
- Breath markers every 2–4 bars for performance realism.

## Quality Checklist
- Does cadence evolve every 4 bars?
- Are punchlines rhythmically framed?
- Is articulation feasible at target BPM?
