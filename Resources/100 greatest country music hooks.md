# 100 Greatest Country Hooks — LLM Reference Companion

## Purpose
Provide hook-level writing patterns extracted from classic and modern country standards, so LLM outputs can emulate proven memorability principles.

## What to Learn from the List
1. **Conversational simplicity** beats ornate phrasing.
2. **Title repetition** increases recall when melody contour varies.
3. **Emotional plain truth** outperforms clever abstraction.
4. **Specific details** (times, places, objects) make hooks believable.
5. **Prosody fit** (stress aligned with strong beats) is non-negotiable.

## Hook Construction Patterns
- Direct plea: "Don’t leave" style emotional immediacy.
- Paradox line: conflicting truth that sparks curiosity.
- Place-based identity: town, road, bar, county image.
- Life thesis: one-line philosophy with sing-along shape.

## LLM Usage Pattern
- Ask for 20 title hooks.
- Classify by pattern type above.
- Select top 5 by memorability and emotional clarity.
- Expand only the best candidate into full chorus.

## Quality Gate
Reject hooks that fail any of the following:
- Too abstract
- Too long to sing in one breath
- No clear title phrase
- No emotional point of view
