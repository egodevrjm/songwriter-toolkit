# Hip-Hop Structure Playbook

## Objective
Balance replayability (strong hooks) with narrative momentum (verses that progress).

## Common Form
- Intro: 4–8 bars
- Hook: 8 bars
- Verse 1: 16 bars
- Hook: 8 bars
- Verse 2: 16 bars
- Hook: 8 bars
- Bridge / breakdown / feature: 8–16 bars
- Final hook/outro: 8–16 bars

## Structural Principles
- Start quickly; dead intros reduce retention.
- Make each verse add new perspective, stakes, or detail.
- Use beat switches or dropouts only when they serve the story.

## LLM Planning Guidance
Ask for:
- Bar map with cadence intent by section.
- Hook wording variants for repetition without staleness.
- Optional feature verse placement logic.
- Performance notes (ad-libs, doubles, call-response moments).

## Quality Checklist
- Does Verse 2 reveal something new?
- Is the hook phrasing short enough for memorability?
- Are section lengths consistent with intended platform/runtime?
