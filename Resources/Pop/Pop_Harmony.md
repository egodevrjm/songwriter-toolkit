# Pop Harmony Playbook

## Mission
Build harmonic loops that support topline hooks and maintain replay value.

## Genre DNA
- Functional, familiar progressions outperform novelty for mainstream hooks.
- Chorus usually benefits from harmonic stabilization, not complexity spikes.

## Best Practices
- Start with one of: I–V–vi–IV, vi–IV–I–V, I–vi–IV–V.
- Reserve non-diatonic moments for section transitions.
- Align chord changes with lyric stress points.
- Prefer clear bass motion over dense chord extensions.

## LLM Workflow
- Request 3 progression families per concept (anthemic, wistful, driving).
- Generate section-specific chord durations in bars.
- Add one tasteful substitution option per section.
- Return MIDI note suggestions for comp layers.

## Anti-Patterns
- Over-modulation that weakens hook identity.
- Constant reharm under vocal-heavy sections.

## Evaluation Rubric
- Hook support (1-10)
- Section contrast clarity (1-10)
- Arrangement friendliness (1-10)
## Objective
Use familiar harmonic motion to support top-line hooks without distracting from melody and lyric.

## Core Principles
- Favor loop-based progressions that establish identity quickly.
- Let harmonic rhythm serve vocal phrasing (chord changes at lyrical pivots).
- Keep choruses harmonically stable unless contrast is the hook.

## Common Progressions
- I–V–vi–IV: modern anthemic default.
- I–vi–IV–V: nostalgic, doo-wop lineage.
- vi–IV–I–V: moodier verse loops.
- I–♭VII–IV: pop-rock/Mixolydian color.

## LLM Composition Guidance
Prompt the model to output:
- Key, tempo range, and progression per section.
- Chord durations in bars.
- A contrast plan (e.g., minor verse → brighter chorus).
- One substitution option per section (secondary dominant, borrowed chord, or slash bass).

## Quality Checklist
- Is the chorus progression simpler than or equal to the verse?
- Do melodic high points land on stable chord tones?
- Is harmonic contrast purposeful, not random?
