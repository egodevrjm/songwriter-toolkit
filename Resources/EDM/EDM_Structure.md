# EDM Structure Playbook

## Mission
Engineer DJ-friendly structures with predictable phrasing and high-impact release moments.

## Baseline Form
Intro (16-32) → Theme (16) → Build (8-16) → Drop (16-32) → Breakdown (8-16) → Build 2 → Drop 2 → Outro (16).

## Best Practices
- Keep phrase lengths in 8-bar multiples.
- Make second drop an escalation, not a copy.
- Use breakdown to reset frequency and emotional density.

## LLM Workflow
- Generate bar-accurate arrangement with FX cues.
- Output radio edit (<3:30) and extended DJ mix (>5:00).
- Add transition risk checks for live mixing.

## Evaluation Rubric
- Build-to-drop payoff
- DJ usability
- Drop differentiation
## Objective
Engineer tension-and-release cycles that support DJ usability and audience impact.

## Typical Form
- Intro (DJ-friendly): 16–32 bars
- Theme/verse: 16 bars
- Build: 8–16 bars
- Drop: 16–32 bars
- Breakdown: 8–16 bars
- Build 2: 8–16 bars
- Drop 2 / final: 16–32 bars
- Outro: 16 bars

## Structural Principles
- Keep section lengths in 8-bar multiples for mixing.
- Treat each drop as a distinct emotional chapter.
- Use breakdowns to reset listener ear fatigue.

## LLM Planning Guidance
Request:
- Bar-accurate arrangement with automation cues.
- FX lane plan (uplifters, downlifters, impacts, fills).
- Alternate radio edit structure under 3:30.
- DJ extended mix structure over 5:00.

## Quality Checklist
- Is there enough contrast between build and drop?
- Are intros/outros usable for transitions?
- Does second drop escalate meaningfully?
