# Pop Structure Playbook

## Mission
Design efficient emotional arcs optimized for streaming retention and chorus recall.

## Baseline Form
Intro (4-8) → Verse (8-16) → Pre (4-8) → Chorus (8) → Verse → Pre → Chorus → Bridge (8) → Final Chorus/Outro.

## Best Practices
- Deliver core hook before 0:50.
- Increase energy each chorus return (stacking, ad-libs, percussion lift).
- Bridge should introduce new emotional information, not filler.
- Keep transitions active (pickup vocal, drum fill, reverse swell).

## LLM Workflow
- Request a bar-by-bar blueprint with energy level per section.
- Add "radio edit" and "extended version" options.
- Ask for transition devices per section boundary.

## Anti-Patterns
- Long intros without identity cues.
- Identical chorus arrangement every repeat.

## Evaluation Rubric
- Retention potential in first 30s
- Narrative progression across verses
- Final third payoff strength
