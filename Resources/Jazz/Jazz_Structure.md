# Jazz Structure Playbook

## Mission
Use form as a shared map for improvisation, interplay, and dynamic storytelling.

## Common Forms
- 12-bar blues
- 32-bar AABA
- ABAC / modal vamps

## Best Practices
- Present head clearly before exploratory solos.
- Shape solo order for dynamic contour.
- Plan re-entry and ending (tag, vamp, hard stop).

## LLM Workflow
- Request chorus-by-chorus arrangement plan.
- Add cues for trading sections and backgrounds.
- Produce short and extended set versions.

## Evaluation Rubric
- Form clarity
- Dynamic arc quality
- Ensemble cue precision
